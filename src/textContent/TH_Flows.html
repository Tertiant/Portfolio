Continuing from the interviews, surveys, and personas there are five core flows that the website needs to support:

  1. Users need to be able to search for cards
  2. Users should be able to filter results based on card attributes like condition, foiling, and art
  3. Users should be able to add cards to a watchlist
  4. Users should be able to view a card's details
  5. Users should be able to buy any number of cards directly through the website
